【文件说明】

一、核心算法文件
-----------------

1. thumb_ik_q1q2.m
   功能：求解 q1 和 q2（约束平面姿态）
   输入：P_target (末端位置), phi (平面夹角)
   输出：q1, q2, info (包含法向量、求解状态等)
   方法：混合容差策略（解析法 + 搜索法）
   
   数学原理：
   - 约束方程 1: cos(q1)·cos(q2) = cos(φ)
   - 约束方程 2: -x·sin(q2) + y·cos(q1)·cos(q2) + z·sin(q1)·cos(q2) = 0
   - 方法：联立消元，化为一元二次方程求解

2. thumb_ik_q3q4.m
   功能：求解 q3 和 q4（平面内 2R 逆解）
   输入：P_target (末端位置), q1, q2 (已由上一步求得)
   输出：q3, q4, info (包含 M/N 点位置、中间变量等)
   方法：几何法解析解（余弦定理 + 三角关系）
   
   数学原理：
   - 步骤 1: 计算 MT 距离 r 和方向角 β
   - 步骤 2: 余弦定理求 q4（注意是外角）
   - 步骤 3: 余弦定理求∠NMT
   - 步骤 4: q3 = β ∓ ∠NMT - α2

3. thumb_fk_main.m
   功能：正运动学求解（根据关节角度计算末端位置和姿态）
   输入：q1, q2, q3, q4 (关节角度)
   输出：T_end (齐次变换矩阵), P_end (位置), R_end (姿态), info (中间信息)
   方法：DH 参数法 + 齐次变换

二、参数文件
-----------

4. params.m
   功能：定义正逆解所需的所有结构常数和配置参数
   内容：
   - 结构常数：alpha1, alpha2, alpha3, L1, L2, L3
   - 关节限位：q1~q4 的最小/最大值
   - 容差参数：位置容差、角度容差、混合策略阈值
   - 验证容差：位置验证、共面性验证、d_y 验证
   
   说明：修改此文件中的参数将同时影响正解和逆解计算

三、调试/测试脚本
-----------------

5. thumb_ik_solve.m
   功能：完整逆解求解测试脚本
   说明：调用 thumb_ik_q1q2 和 thumb_ik_q3q4，完成完整逆解并用正解验证
   用途：调试逆解算法，验证求解正确性

6. thumb_fk_solve.m
   功能：正运动学求解测试脚本
   说明：调用 thumb_fk_main，验证正解计算和共面性
   用途：调试正解算法，验证机构运动学特性

四、可视化模块（visualization 文件夹）
---------------------------------------
详细说明请参阅 visualization/README_vis.md

7. vis_params.m
   功能：定义所有可视化配置参数
   内容：视频参数、颜色定义、窗口设置、运动阶段参数等

8. thumb_vis_joint_traj.m
   功能：生成关节轨迹序列并计算正解
   输出：vis_joint_traj_data.mat

9. thumb_vis_ik_solve.m
   功能：批量逆解求解并分析误差
   输出：vis_ik_solve_data.mat

10. thumb_vis_generate_points.m
    功能：生成可视化坐标（O/M/N/T 点）
    输出：vis_points_data.mat

11. thumb_vis_animation_3d.m
    功能：生成 3D 动画视频
    输出：thumb_ik_animation_3d.avi

12. thumb_vis_diagnose_failures.m
    功能：诊断分析逆解失败帧
    使用时机：逆解失败率 > 5% 时使用

【使用流程】

一、逆解求解
-----------
1. 修改 thumb_ik_solve.m 中的输入参数：
   P_target = [x; y; z];   % 末端目标位置 (mm)
   phi = 角度值;            % 约束平面夹角 (度)

2. 运行脚本：
   >> thumb_ik_solve

3. 查看输出：
   - 关节角度 q1, q2, q3, q4
   - 位置误差（正解验证）
   - 可视化图形（如启用）

二、正解求解
-----------
1. 修改 thumb_fk_solve.m 中的关节角度：
   q1 = 角度值; q2 = 角度值; q3 = 角度值; q4 = 角度值;

2. 运行脚本：
   >> thumb_fk_solve

3. 查看输出：
   - 末端位置 P_end
   - 末端姿态 R_end
   - 共面性验证结果

三、可视化流程
-------------
1. 进入 visualization 文件夹：
   >> cd visualization

2. 按顺序运行脚本：
   >> thumb_vis_joint_traj      % 步骤 1: 生成关节轨迹
   >> thumb_vis_ik_solve        % 步骤 2: 逆解求解
   >> thumb_vis_generate_points % 步骤 3: 生成可视化坐标
   >> thumb_vis_animation_3d    % 步骤 4: 生成 3D 动画视频

3. 播放视频：
   >> implay('thumb_ik_animation_3d.avi')

四、参数修改
-----------
1. 修改 params.m 中的结构常数或容差参数
   - 影响：正解和逆解计算

2. 修改 vis_params.m 中的可视化参数
   - 影响：仅可视化效果

================================================================================

【参数说明】

结构常数（与正解保持一致）：
  alpha1 = -3.65°     基座固定偏角
  alpha2 = 8.22°      M 系固定偏角
  alpha3 = -16.72°    N 系固定偏角
  L1 = 23.56 mm       OM 长度
  L2 = 53.17 mm       MN 长度
  L3 = 25.57 mm       NT 长度

关节限位：
  q1: -95° ~ 0°
  q2: -85° ~ 0°
  q3: -95° ~ 10°
  q4: -80° ~ 8°

容差参数：
  tol_x, tol_y, tol_z: 0.02 mm (位置容差)
  tol_phi: 0.1° (平面夹角容差)
  limit_tolerance: 0.5° (限位检查容差)
  position_tolerance: 0.01 mm (位置验证容差)

================================================================================

【错误代码说明】

info.status:
  0 = 求解成功
  1 = 无解
  2 = 多解已筛选/警告

常见错误信息：
  - "判别式 delta < 0"：目标点超出工作空间
  - "所有候选解均超出关节限位"：目标点可达但关节角度超出物理限制
  - "phi 超出 q2 可行范围"：平面夹角与关节限位矛盾
  - "目标点超出最大/最小工作空间"：MT 距离超出 L2+L3 或小于 |L2-L3|

================================================================================

【注意事项】

1. phi 的取值范围为 [0°, 90°]，由关节限位决定法向量 Y 分量恒为正

2. d_y 验证：MT 向量在{1}系中的 Y 分量理论值为 0
   如果 |d_y| > 1e-6，说明 q1/q2 求解存在数值误差

3. 多解处理：当存在多组可行解时，默认选择关节角绝对值和最小的解

4. 参数一致性：
   - 修改 params.m 中的参数后，所有正逆解计算将自动更新
   - 修改 vis_params.m 中的参数后，所有可视化效果将自动更新

5. 文件依赖关系：
   
   核心算法：
   thumb_ik_solve.m → thumb_ik_q1q2.m → params.m
                      thumb_ik_q3q4.m → params.m
                      thumb_fk_main.m → params.m
   thumb_fk_solve.m → thumb_fk_main.m → params.m
   
   可视化模块：
   thumb_vis_joint_traj.m → thumb_fk_main.m → params.m
                            vis_params.m
   thumb_vis_ik_solve.m → thumb_ik_q1q2.m → params.m
                          thumb_ik_q3q4.m → params.m
   thumb_vis_generate_points.m → thumb_fk_main.m → params.m
   thumb_vis_animation_3d.m → vis_params.m
   thumb_vis_diagnose_failures.m → (仅数据分析)
